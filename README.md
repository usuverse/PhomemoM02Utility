Default settings should be:<br>
Print width - 384<br>
Dither - Floyd-Steinberg (serpentine)<br>
Threshold - 128<br>
Gamma - 50%<br>
Chunk size - 160<br>
Band rows - 0<br>
Inter-band delay - 0<br>
Inter-chunk delay - 0<br>
<br>
If printing something long, might need to mess with band rows, interband, interchunk.<br>
