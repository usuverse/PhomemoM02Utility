Default settings should be:
Print width - 384
Dither - Floyd-Steinberg (serpentine)
Threshold - 128
Gamma - 50%
Chunk size - 160
Band rows - 0
Inter-band delay - 0
Inter-chunk delay - 0

If printing something long, might need to mess with band rows, interband, interchunk.
